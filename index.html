Improving The Gameplay Quality of Valheim

Check out a few our features below, you can also find a full detailes list here.

Advanced Building

Players can modify a objects position and rotation in the default building mode and our addtional modes.

Advanced Editing

Players can select an existing item by pressing the hotkey, they can then modify the objects position and rotation.

Shared Maps

Players can now share the discovered map whether they are online or offline, it syncs to the server upon joining.

Change Server Slots

Part of a large clan? We make it possible for you to adjust your server slots to any size and even remove the password!

How does the Installation Work?

You're only a few steps away

1. Download V+

Download the mod and its required dependencies from this page for your respective operation system and needs.

2. Install V+

Install valheim plus on your client and on your server if you are using a dedicated server. You can follow these detailed instruction on how to.

3. Configuration

Change the values, settings and features to your liking inside the provided valheim_plus.cfg. Instructions about that here.
